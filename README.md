### 👨‍💻 About Me

I am a **Full Stack Developer**, **AI/ML Enthusiast**, and **Tech Entrepreneur** based in Mauritania. I specialize in building scalable SaaS solutions, training AI models, creating datasets, and have contributed to critical national digital infrastructures.

- 🔭 **Focus**: Digitizing the Mauritanian ecosystem through specialized SaaS platforms, AI/ML solutions, and mobile applications.
- 💼 **Experience**: GovTech, Cyber Security Systems, B2B SaaS, AI/ML Model Training, Dataset Creation.
- 🧠 **Learning**: Advanced MLOps, Cyber Security protocols, and cutting-edge AI/ML techniques.
- 🤖 **AI/ML**: Training AI models, working with datasets, creating and managing datasets for machine learning projects.
- 📱 **Mobile**: Building cross-platform mobile applications with React Native, Expo, and Flutter.
- 🌱 **Always Learning**: Continuously exploring new technologies in web development, mobile development, and artificial intelligence.
